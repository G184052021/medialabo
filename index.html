「Webコンテンツ作成」の成果

本テーマの学習目的

HTML, CSS, JavaScript の基本を学び，これらを組み合わせて Web ページを構成する方法を学びます。HTML による Web ページの構造の記述、CSS による Web ページの装飾とJavaScript の基本を学びます。HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法を分かります。HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせて簡単な Web ページが作成できることを勉強します。

プロジェクト課題について

プロジェクト課題ではWebコンテンツの作成を実施します。まず、三種類のデータのうち，世界の天気のデータの Web アプリケーションを作るか決定しました。データは次の三種類として，そのいずれかのデータを検索，結果表示できるようにします．データの詳細は参考資料 に掲載しています。検索結果の HTML を作っていました。また、ページを装飾するのCSSを作成しました。天気予報のサンプルデータを含む JavaScript プログラムのテンプレートを用意しました。このデータの一部の情報をコンソールに出力するプログラムを作っていました。次は、JavaScript プログラムを改良して，ページ上にデータを表示しました。最後、検索用の input 要素やラベル，ボタンを設置しました。

第6回Webページ作成演習

工夫点，苦心点，強調点，感想

インターネットの急速な発展に伴い、ウェブデザインは急速に発展してきました。インターネットアクセスの主なサポートとして、人々がインターネットを頻繁に使用するため、ウェブデザインは非常に重要になっています。 オンラインで毎日ウェブページに出くわし、ウェブページのデザインには気を配っていませんでしたが、この時期に学び、理解した結果、ウェブデザインにも多くの問題を考慮する必要があることに気づきました。

先生の説明や教科書を読むことで、ウェブサイトの構築やウェブページの作成についてある程度の知識を身につけ、ウェブページの作成の基礎知識もある程度理解しています。 Visual StudioCodeを使用してWebページを作成し、いくつかの基本的な機能を学びました。

Web ページはハイパーリンクによって Web ページが接続されていて，ユーザはリンクを辿ってたくさんの Web ページを見ることができます。Web ページではテキストに加えて，画像や音声，動画などマルチメディアコンテンツや PDF などのファイルにアクセスできます。Web ページのサーバに接続して，サーバが提供するサービスを利用できます。Web ページを構成する「部品」は大きく4つに分けられます。HTMLはWeb ページの構造を指定するための記述です。ページは見出しや本文，箇条書き，図表などでできていますが，このようなページの構造を記述するのが HTMLです。CSSはWeb ページの見た目を指定するための記述です。JavaScriptはWeb ページの動作を指定するプログラムです。Web ページには画像や音声，動画などを貼り付けてユーザに提供することができます。

基礎知識は身につけましたが、自分でやろうとするといつも問題がたくさんあります。 私はこれまでウェブデザインに触れたことがないので、学習プロセスのすべてが完全に新しいものです。 多くのウェブデザイン用語は理解するのが難しい場合があります。しかし、学習の過程で、教室での演習の実践を通して、知識のポイントをよりよく理解することができます。 問題がある場合、授業で説明されていたとしても、理解できない点がいくつかあり、先生がもう一度説明してくれます。特に数当てゲームの課題では、数当てゲームの結果をコンソールではなく，ページに表示するように修正することは難しいと思います。しかし、先生の助けを借りて、最終的に無事に完了しました。

Webコンテンツ作成の勉強を通して、基本的なウェブデザインについてある程度理解していますが、まだ欠点があり、研究と改善を続ける必要があります。
